# Book of Truth

Name: Minhua Ren  
Date: 2026.3.12  

# Overview
This project presents a “Book of Truth” that provides real-time advice and activity suggestions through a digital interface.  
Users can receive simple guidance for dealing with chaotic or disordered moods in daily life. The system combines randomness and interaction to create a playful and engaging experience.

# How It Works
1. Different pieces of advice are encoded as numbers to build a dynamic sorting system.
2. The system calls a dedicated HTTP API to retrieve suggestions in real time.
3. Each time the user interacts with the interface, the advice appears in a different order.

# Interaction
Users interact with the digital interface to receive suggestions.  
Because the advice is dynamically generated and reordered, each interaction produces a slightly different result, reflecting the unpredictable and playful nature of daily life.
